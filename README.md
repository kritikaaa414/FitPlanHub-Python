# FitPlanHub-Python

## Project Overview
FitPlanHub is a simple full-stack web application built using Python and Flask.
The project demonstrates backend API development, database integration,
and a basic frontend using HTML and JavaScript.

This project was developed as part of an assignment to showcase
full-stack development fundamentals.

---

## Features Implemented
- User Signup and Login
- Role-based user handling
- Workout / Plan management
- Backend REST APIs using Flask
- Database integration using SQLite
- Basic frontend using HTML and JavaScript
- CRUD operations (Create, Read, Update, Delete)

---

## Tech Stack Used
- Backend: Python (Flask)
- Frontend: HTML, JavaScript
- Database: SQLite
- Version Control: Git & GitHub
